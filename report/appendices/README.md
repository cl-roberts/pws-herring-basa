# Generate appendices for report

The `write-appendix.bat` batch file is a simple pandoc wrapper to render markdown files in `docs/` to appendices to be used in the annual BASA report. Execute the batch file to generate the appendices.

```
write-appendix.bat
```

## `formulation.md` 

This is a set of tables that give the full, current formulation of BASA.

## `forecast.md` 

This explains how the annual spring pre-fishery mature biomass forecast is calculated
